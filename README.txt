This code is programmed to run in windows, to run the game you should first set up the server using the server.cpp file.
run server.cpp using the following code in your terminal:

g++ -o server server.cpp -lws2_32

After setting up the server you'll be able to add players running the client.py code. Add at least 2 players and enter their usernames in the terminal and wait for the game to matchmake.
